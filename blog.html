How I made this project

This is the blog part of the web! To show you the process of the web being made, with its ups and downs. I wanted to make a website, with my idea, not a blog or an online shop that did nothing. I also wanted to make it a fun website; I was given freedom and planned to use it to make something unserious and enjoyable; it didn’t need to be too complicated. So, I came up with the idea you just saw, a website that has nothing but a rating for you to “choose” how good the website was, though the button didn’t allow anything but a 10/10, and I intentionally made it dull and without an ounce of decoration, since it was part of the joke. It was creative and a joke, but it still was a lot more complicated than I thought it would be. At first, we all tried WordPress of course, and it worked great on theory, had the personalized html block I needed to make the website! But it did not work... I wondered why, and after an afternoon spent trying to figure out why the code wasn’t working out, I realized it wanted me to pay to have it my way, and so I decided that I didn’t want to make another web, I had already committed a whole afternoon! Still, I thought that if it did not work out, I would make a website blog about my hardships, but before gving up I searched other webs that could work! There were lots of options, but most didn’t work as they used templates, they were simply meant for the bare minimum unless you paid, and I am a broke and stingy teenager, so I talked with ChatGPT, who by the way was the one to also help me code everything from the start! Since well you know, I do not have the ability to code, but you did give me permission when I told you, as long as it was still my web, and ChatGPT only made the coding. Very complicated plan(fail) After some time, I’m not sure if ChatGPT thought I wanted a very professional and functioning web, but we came up with a very convoluted and complicated plan, in which I used render, then also GitHub pages and some other webs to make it all very manually. Still as before, I was only able to get so far with the plan, as I am not a programmer. Then the one name came back to my head, GitHub Pages. I asked the unpaid coder I had (ChatGPT), and he said that it was possible to make my web only on GitHub pages! That made the plan a lot simpler, and after creating a repository, I was able to create my index file and make the code! Coding itself was also hard, not because of me having to do it, as previously stated I had a slave coworker in this, and I could ask him to do all the work, but when he added something, another thing crashed... And I spent two afternoons trying to make only the initial idea and not this blog! (at the time of writing this, it has not yet been incorporated) So let us hope you see this in a functional web after having clicked the correct grade! Well, this is where we are now. Hopefully selecting number 10 gave you a new internal link to this blog, and that I implemented external links on the websites I tried to use. Because I clearly spent more time on this, and I do not want to fight with ChatGPT anymore. Update! It took me two afternoons, but I finished (unless you give me another thing to add) This was a looong blog! But this marks the end of it, so if you did read this whole thing, congratulations to you, and I hope you liked my web! (the button will not let you dislike it anyway)
